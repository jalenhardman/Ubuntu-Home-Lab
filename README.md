## Ubuntu Home Lab (UTM)

Built a basic Ubuntu virtual machine using UTM on Apple Silicon.

### What I did
- Installed Ubuntu 24.04 LTS (ARM64)
- Verified user access with whoami
- Checked network configuration using ip a
- Confirmed internet connectivity with ping

### Why this matters
This lab demonstrates foundational skills in:
- Virtualization
- Linux basics
- Network connectivity
- SOC-style documentation

Screenshots and notes are included as evidence in a separate folder.
